你好，我是蔡英文

民進黨是一個「來自人民、為了人民」的政黨，儘管沒有豐厚的黨產，我們仍然堅持以小額募款來籌措政黨財源。自從2012年「三隻小豬」的熱潮以來，小額募款也漸漸成為台灣社會視為理所當然的政治原則。

人民支持的力量，使我們能夠拒絕不健康的政商關係、減少日後施政的包袱。因此，這一次參選，我也將繼續秉持小額募款的原則，並將公開選舉經費支出，向人民負責。

選舉並不只在投票日，過程中的每一天，都是價值的選擇。我做出了這樣的選擇，你也可以透過小額捐款，作出你的選擇。我們的資源並不豐厚，志工的一頓午飯、一疊政策說帖、幾箱礦泉水，都能聚沙成塔、累積成巨大的改變。

每一天，我們都要再努力一點點，一起點亮台灣

捐贈者應注意事項：

  1. 單筆捐款金額如超過新台幣一萬元整，請捐款人務必提供完整資料。如未能提供資料，致使無法開立政治獻金收據者，依政治獻金法規定 該筆捐款需全數繳國庫
  2. 有以下情形之個人、公司、團 體，無法接受捐贈：
    • 政府或公營事業持有資本達百分之二十之民營企業
    • 與政府機關有巨額採購之廠商
    • 有累積虧損尚未依規定彌補之營利事業
    • 未具選舉權之人
    • 宗教團體
    • 外國、大陸地區及港澳地區之人民、法人、團體或其它機構
  3. 每年對本黨捐贈金額上限：
    • 個人：新台幣三十萬（可申報列舉扣除額之上限為二十萬元）
    • 營利事業：新台幣三百萬元（可申報當年度費用或損失之上限為五十萬元）
    • 人民團體：新台幣二百萬元
  4. 其它規定請參閱政治獻金法或監察院網站
